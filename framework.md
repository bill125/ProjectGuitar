# Loop

分工：L

用状态机设计：两种状态：未录制、录制

输入：

1. 开始\结束
2. 录音编号（事先在SRAM中分配3段内存用于存储）
3. 音高
4. 乐器编号

输出：

1. 旋律


将整体旋律写入sram
